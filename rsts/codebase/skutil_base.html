skutil.base¶

class skutil.base.SamplingWarning[source]¶

Bases: exceptions.UserWarning

Custom warning used to notify the user that sub-optimal sampling behavior has occurred. For instance, performing oversampling on a minority class with only one instance will cause this warning to be thrown.

class skutil.base.SelectiveWarning[source]¶

Bases: exceptions.UserWarning

Custom warning used to notify user when a structure implementing SelectiveMixin operates improperly. A common usecase is when the fit method receives a non-DataFrame X, and no cols.

class skutil.base.ModuleImportWarning[source]¶

Bases: exceptions.UserWarning

Custom warning used to notify user a non-critical import failed, and to suggest the installation of the module for optimal results.

class skutil.base.SelectiveMixin[source]¶

A mixin class that all selective transformers should implement. All SelectiveMixin implementers should only apply their fit method on the defined columns.
